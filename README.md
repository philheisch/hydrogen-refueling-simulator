
# Hydrogen Refueling Simulator

This simulator helps design and test hydrogen refueling stations for vehicles at 350 bar or 700 bar.

## Features

- Input vehicle count, tank size, starting pressure
- Configure compressor type and nozzle count
- Calculate H2 need per hour
- Estimate number of compressors and nozzles
- Display fill curve (SAE-type)

## How to Run

```bash
streamlit run hydrogen_refueling_simulator.py
```
